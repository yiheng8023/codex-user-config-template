# Security Policy

## Reporting

If you find a security issue in this template, please open a private report through GitHub security advisories when available, or contact the repository owner privately.

Do not open a public issue containing secrets, tokens, personal memory, private paths, or account details.

## Supported scope

This repository contains public-safe scaffolding only. It does not contain live credentials, OAuth state, private memory, runtime logs, or a user's actual Codex configuration.
